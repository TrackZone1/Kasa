# Kasa Front

Le projet front utilise maintenant Vite pour le développement, le build et les tests.

## Scripts disponibles

Dans le dossier front, tu peux lancer :

### `npm run dev`

Démarre le serveur de développement Vite.

### `npm run build`

Génère le build de production dans le dossier `dist`.

### `npm run preview`

Lance un serveur local pour prévisualiser le build de production.

### `npm test`

Exécute les tests avec Vitest dans un mode non interactif.

## Données locales

Les données de démonstration restent servies depuis `public/fakeData/logements.json` et sont chargées via `/fakeData/logements.json`.
